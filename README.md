## Getting Started

#### Clone the repo

`$ git clone git@github.com:brianseitel/node-wars.git`

#### Install Dependencies

`$ npm install`

#### Set Configuration

Update config/game.yml to reflect your desired values. See the wiki for more information.


#### Kick off Big Bang

Run `$ node init.js`

#### Start game

Run `# node index.js`
